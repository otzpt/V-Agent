V-Agent Assets Folder
=====================

Place custom icons and graphics here:

vagent.ico
  - Main application icon (256x256 recommended)
  - Used in build.bat and setup.iss
  - Format: Windows Icon (.ico)
  
To create an icon from a PNG:
  1. Create a 256x256 PNG image
  2. Use an online converter: https://convertio.co/png-ico/
  3. Save as vagent.ico in this folder
  4. Run build.bat to use it in the installer

If vagent.ico is not present, the installer will use a default icon.
